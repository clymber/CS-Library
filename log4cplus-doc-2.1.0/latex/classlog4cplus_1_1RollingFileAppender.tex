\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlog4cplus_1_1RollingFileAppender}{Rolling\+File\+Appender}} extends \mbox{\hyperlink{classlog4cplus_1_1FileAppender}{File\+Appender}} to backup the log files when they reach a certain size. 

\doxyparagraph*{Properties}

Properties additional to {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1FileAppender}{File\+Appender}}}\textquotesingle{}s properties\+:


\begin{DoxyDescription}
\item[{\ttfamily Max\+File\+Size} ]This property specifies maximal size of output file. The value is in bytes. It is possible to use {\ttfamily MB} and {\ttfamily KB} suffixes to specify the value in megabytes or kilobytes instead.


\item[{\ttfamily Max\+Backup\+Index} ]This property limits the number of backup output files; e.\+g. how many {\ttfamily log.\+1}, {\ttfamily log.\+2} etc. files will be kept.
