\doxysubsection{Detailed Description}
This is a simple filter based on the key/value pair stored in \mbox{\hyperlink{classlog4cplus_1_1MDC}{MDC}}. 

The filter admits four options {\bfseries{Neutral\+On\+Empty}}, {\bfseries{MDCKey\+To\+Match}} {\bfseries{MDCValue\+To\+Match}} and {\bfseries{Accept\+On\+Match}}.

If {\ttfamily Neutral\+On\+Empty} is true and {\ttfamily MDCKey\+To\+Match} or {\ttfamily MDCValue\+To\+Match} is empty then {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6ae75ab2e37a542c14fe57be606502a550}{NEUTRAL}}} is returned.

If {\ttfamily Neutral\+On\+Empty} is true and the string returned by event.\+get\+MDC(\+MDCKey\+To\+Match) is empty then {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6ae75ab2e37a542c14fe57be606502a550}{NEUTRAL}}} is returned.

If the string returned by event.\+get\+MDC(\+MDCKey\+To\+Match) matches {\ttfamily MDCValue\+To\+Match}, then if {\bfseries{Accept\+On\+Match}} is true, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a222b6ce4c30f1d7d9164b901bb2907a7}{ACCEPT}}} is returned, and if {\bfseries{Accept\+On\+Match}} is false, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a4782fd85324c15b37a4de87628f51634}{DENY}}} is returned.

If the string returned by event.\+get\+MDC(\+MDCKey\+To\+Match) does not match {\ttfamily MDCValue\+To\+Match}, then if {\bfseries{Accept\+On\+Match}} is true, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a4782fd85324c15b37a4de87628f51634}{DENY}}} is returned, and if {\bfseries{Accept\+On\+Match}} is false, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a222b6ce4c30f1d7d9164b901bb2907a7}{ACCEPT}}} is returned.
