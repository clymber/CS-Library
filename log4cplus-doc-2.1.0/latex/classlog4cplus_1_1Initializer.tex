\doxysubsection{Detailed Description}
This class helps with initialization and shutdown of \mbox{\hyperlink{namespacelog4cplus}{log4cplus}}. 

Its constructor calls {\ttfamily log4cplus\+::initialize()} and its destructor calls {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1Logger_a9a288b93105766e0ad7bebbbf8c5ea2f}{log4cplus\+::\+Logger\+::shutdown()}}}. Use this class as the first thing in your {\ttfamily main()}. It will ensure shutdown of \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} at the end of {\ttfamily main()}. This is particularly important on Windows, where shutdown of standard threads outside {\ttfamily main()} is impossible.
