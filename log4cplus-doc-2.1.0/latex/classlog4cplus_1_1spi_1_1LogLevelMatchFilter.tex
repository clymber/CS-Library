\doxysubsection{Detailed Description}
This is a very simple filter based on Log\+Level matching. 

The filter admits two options {\bfseries{Log\+Level\+To\+Match}} and {\bfseries{Accept\+On\+Match}}. If there is an exact match between the value of the Log\+Level\+To\+Match option and the Log\+Level of the {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1spi_1_1InternalLoggingEvent}{spi\+::\+Internal\+Logging\+Event}}}, then the {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1spi_1_1LogLevelMatchFilter_a69cb65938f76fc61a5fd2f16172a9d33}{decide}}} method returns {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a222b6ce4c30f1d7d9164b901bb2907a7}{ACCEPT}}} in case the {\bfseries{Accept\+On\+Match}} option value is set to {\ttfamily true}, if it is {\ttfamily false} then {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a4782fd85324c15b37a4de87628f51634}{DENY}}} is returned. If there is no match, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6ae75ab2e37a542c14fe57be606502a550}{NEUTRAL}}} is returned.
