\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlog4cplus_1_1ConsoleAppender}{Console\+Appender}} appends log events to {\ttfamily std\+::cout} or {\ttfamily std\+::cerr} using a layout specified by the user. 

The default target is {\ttfamily std\+::cout}.

\doxyparagraph*{Properties}


\begin{DoxyDescription}
\item[{\ttfamily log\+To\+Std\+Err} ]When it is set true, the output stream will be {\ttfamily std\+::cerr} instead of {\ttfamily std\+::cout}.


\item[{\ttfamily Immediate\+Flush} ]When it is set true, output stream will be flushed after each appended event.


\item[{\ttfamily Locale} ]This property specifies a locale name that will be imbued into output stream. Locale can be specified either by system specific locale name, e.\+g., {\ttfamily en\+\_\+\+US.\+UTF-\/8}, or by one of four recognized keywords\+: {\ttfamily GLOBAL}, {\ttfamily DEFAULT} (which is an alias for {\ttfamily GLOBAL}), {\ttfamily USER} and {\ttfamily CLASSIC}. When specified locale is not available, {\ttfamily GLOBAL} is used instead. It is possible to register additional locale keywords by registering an instance of {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1spi_1_1LocaleFactory}{spi\+::\+Locale\+Factory}}} in {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_ae93cf5d075d4355919917879db043004}{spi\+::\+Locale\+Factory\+Registry}}}. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{namespacelog4cplus_1_1spi_a791f20ebec905565a1cea549b905b9a7}{spi\+::get\+Locale\+Factory\+Registry()}}.
\end{DoxySeeAlso}
Note\+: if {\ttfamily Locale} is set, {\ttfamily Immediate\+Flush} will be set to true automatically.
