\doxysubsection{Detailed Description}
The internal representation of logging events. 

When an affirmative decision is made to log then a {\ttfamily \mbox{\hyperlink{classlog4cplus_1_1spi_1_1InternalLoggingEvent}{Internal\+Logging\+Event}}} instance is created. This instance is passed around to the different \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} components.

This class is of concern to those wishing to extend \mbox{\hyperlink{namespacelog4cplus}{log4cplus}}.
