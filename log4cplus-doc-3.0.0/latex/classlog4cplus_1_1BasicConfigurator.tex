\doxysubsection{Detailed Description}
Use this class to quickly configure the package. 

For file based configuration see \mbox{\hyperlink{classlog4cplus_1_1PropertyConfigurator}{Property\+Configurator}}. \mbox{\hyperlink{classlog4cplus_1_1BasicConfigurator}{Basic\+Configurator}} automatically attaches \mbox{\hyperlink{classlog4cplus_1_1ConsoleAppender}{Console\+Appender}} to {\ttfamily root\+Logger}, with output going to standard output, using DEBUG Log\+Level value. The additional parameter log\+To\+Std\+Err may redirect the output to standard error.
