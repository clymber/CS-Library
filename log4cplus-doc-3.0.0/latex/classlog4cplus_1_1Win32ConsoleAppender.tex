\doxysubsection{Detailed Description}
Prints events to Win32 console. 

\doxyparagraph*{Properties}


\begin{DoxyDescription}
\item[{\ttfamily Alloc\+Console} ]This boolean property specifies whether or not this appender will try to allocate new console using the {\ttfamily Alloc\+Console()} Win32 function.


\item[{\ttfamily log\+To\+Std\+Err} ]When it is set true, the output will be into {\ttfamily STD\+\_\+\+ERROR\+\_\+\+HANDLE} instead of {\ttfamily STD\+\_\+\+OUTPUT\+\_\+\+HANDLE}. 


\item[{\ttfamily Text\+Color} ]See MSDN documentation for \href{http://msdn.microsoft.com/en-us/library/windows/desktop/ms682088(v=vs.85).aspx\#_win32_character_attributes}{\texttt{ Character Attributes}}. 
\end{DoxyDescription}
