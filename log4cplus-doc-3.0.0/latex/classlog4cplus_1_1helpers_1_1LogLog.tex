\doxysubsection{Detailed Description}
This class used to output log statements from within the \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} package. 

Log4cplus components cannot make \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} logging calls. However, it is sometimes useful for the user to learn about what \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} is doing. You can enable \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} internal logging by defining the {\bfseries{log4cplus.\+config\+Debug}} variable.

All \mbox{\hyperlink{namespacelog4cplus}{log4cplus}} internal debug calls go to {\ttfamily cout} where as internal error messages are sent to {\ttfamily cerr}. All internal messages are prepended with the string \char`\"{}log4clus\+: \char`\"{}.
