\doxysubsection{Detailed Description}
This is a simple filter based on the string returned by \mbox{\hyperlink{namespacelog4cplus_a02d988bc9b136a6062ddc303efb6f627}{event.\+get\+NDC()}}. 

The filter admits three options {\bfseries{Neutral\+On\+Empty}}, {\bfseries{NDCTo\+Match}} and {\bfseries{Accept\+On\+Match}}.

If {\ttfamily Neutral\+On\+Empty} is true and {\ttfamily NDCTo\+Match} is empty then {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6ae75ab2e37a542c14fe57be606502a550}{NEUTRAL}}} is returned.

If {\ttfamily Neutral\+On\+Empty} is true and the value returned by \mbox{\hyperlink{namespacelog4cplus_a02d988bc9b136a6062ddc303efb6f627}{event.\+get\+NDC()}} is empty then {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6ae75ab2e37a542c14fe57be606502a550}{NEUTRAL}}} is returned.

If the string returned by \mbox{\hyperlink{namespacelog4cplus_a02d988bc9b136a6062ddc303efb6f627}{event.\+get\+NDC()}} matches {\ttfamily NDCTo\+Match}, then if {\bfseries{Accept\+On\+Match}} is true, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a222b6ce4c30f1d7d9164b901bb2907a7}{ACCEPT}}} is returned, and if {\bfseries{Accept\+On\+Match}} is false, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a4782fd85324c15b37a4de87628f51634}{DENY}}} is returned.

If the string returned by \mbox{\hyperlink{namespacelog4cplus_a02d988bc9b136a6062ddc303efb6f627}{event.\+get\+NDC()}} does not match {\ttfamily NDCTo\+Match}, then if {\bfseries{Accept\+On\+Match}} is true, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a4782fd85324c15b37a4de87628f51634}{DENY}}} is returned, and if {\bfseries{Accept\+On\+Match}} is false, {\ttfamily \mbox{\hyperlink{namespacelog4cplus_1_1spi_aa910f475d36c00f943ef78e37d11e3f6a222b6ce4c30f1d7d9164b901bb2907a7}{ACCEPT}}} is returned.
