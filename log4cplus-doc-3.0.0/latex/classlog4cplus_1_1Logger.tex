Calling this method will {\itshape safely} close and remove all appenders in all the loggers including root contained in the default hierachy. 

Some appenders such as \mbox{\hyperlink{classlog4cplus_1_1SocketAppender}{Socket\+Appender}} need to be closed before the application exits. Otherwise, pending logging events might be lost.

The {\ttfamily shutdown} method is careful to close nested appenders before closing regular appenders. This is allows configurations where a regular appender is attached to a logger and again to a nested appender. \mbox{\Hypertarget{classlog4cplus_1_1Logger_a3073471f478c67e4ebed1c00c8eaa3d4}\label{classlog4cplus_1_1Logger_a3073471f478c67e4ebed1c00c8eaa3d4}}
